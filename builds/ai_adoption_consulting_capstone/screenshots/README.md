# Build 9 — Screenshots Checklist

Screenshots should be captured manually from the running Streamlit app and saved in this folder using the filenames listed below.

## How to Capture

1. Run the app: `streamlit run app.py` from the `ai_adoption_consulting_capstone` directory.
2. Open **http://localhost:8501** in your browser.
3. Navigate to each page via the sidebar radio buttons.
4. Use your browser or OS screenshot tool to capture the full page.
5. Save each screenshot in this folder with the exact filename shown below.

Recommended tool: browser full-page screenshot extension, or macOS `Cmd + Shift + 4` for a selected region.

## Required Screenshots

### 01-capstone-client-setup.png

**Page:** 1. Capstone Client Setup  
**What it demonstrates:** The synthetic capstone client table, cross-build journey stage table, and portfolio indicators. This proves the dashboard is built around a structured evidence model — three client organisations, 21 journey stages across 7 build areas — rather than random demo text.
