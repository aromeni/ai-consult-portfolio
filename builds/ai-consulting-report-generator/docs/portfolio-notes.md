## Longer Portfolio Description

The AI Consulting Report Generator is a Streamlit-based consulting prototype for AI readiness engagements. It starts with a synthetic organisation profile and audit findings for BrightPath Skills Training, then generates a complete consulting workflow: readiness interpretation, risk register, opportunity portfolio, pilot sequence, 30/60/90-day roadmap, report sections, full client report, PDF report, and PowerPoint executive deck.

The system is intentionally deterministic. It does not call external LLM APIs and does not handle real client, learner, HR, safeguarding, personal, confidential, or regulated data. This makes it suitable as a portfolio demonstration of structured AI product thinking, responsible-use design, and consulting workflow automation.

Future productisation would require real-data governance, authentication, audit logging, security review, and client-specific review workflows.
