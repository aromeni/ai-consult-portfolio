# Screenshots — BrightPath AI Readiness + Workflow Audit Tool

This folder holds portfolio and demo screenshots of the running app.

---

## Safety Rules (Read Before Capturing)

**All screenshots must use synthetic BrightPath-style data only.**

- Load BrightPath demo data from the Home page before capturing any screenshot
- Do not capture real learner data, client records, safeguarding information, staff details, or personal data of any kind
- Do not expose local file paths, API keys, credentials, or terminal history containing sensitive information
- Crop browser toolbars, unrelated tabs, and desktop notifications before saving
- Do not commit screenshots containing real client data to any repository
